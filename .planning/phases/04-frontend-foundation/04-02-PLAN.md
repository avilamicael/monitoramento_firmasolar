## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| User input (login form) -> API | Credenciais atravessam do browser para o backend |
| AuthContext state -> UI | Estado de autenticacao controla visibilidade de rotas |
| NavLink -> Router | Navegacao controla qual pagina e renderizada |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-07 | Spoofing | login-form.tsx | mitigate | Credenciais enviadas via HTTPS em prod; em dev via proxy Vite (sem CORS) |
| T-04-08 | Information Disclosure | login-form error | mitigate | Mensagem de erro generica "Credenciais invalidas" — nao expor se email existe ou nao |
| T-04-09 | Tampering | NavUser logout | mitigate | logout() limpa localStorage e reseta estado — usuario nao consegue manter sessao apos logout |
| T-04-10 | Elevation of Privilege | ProtectedLayout bypass | mitigate | ProtectedLayout verifica isAuthenticated no render — manipular URL diretamente nao bypassa a verificacao |
</threat_model>
