```python
class GarantiaUsina(models.Model):
    """
    Garantia comercial de uma usina solar.
    data_fim e ativa sao properties calculadas — nao existem como colunas.
    """
    usina = models.OneToOneField(
        Usina,
        on_delete=models.CASCADE,
        related_name='garantia',
    )
    data_inicio = models.DateField()
    meses = models.PositiveIntegerField()
    observacoes = models.TextField(blank=True)
    criado_em = models.DateTimeField(auto_now_add=True)
    atualizado_em = models.DateTimeField(auto_now=True)

    class Meta:
        verbose_name = 'Garantia de Usina'
        verbose_name_plural = 'Garantias de Usinas'

    @property
    def data_fim(self):
        from dateutil.relativedelta import relativedelta
        return self.data_inicio + relativedelta(months=self.meses)

    @property
    def ativa(self):
        from django.utils import timezone
        return self.data_fim >= timezone.now().date()

    @property
    def dias_restantes(self):
        from django.utils import timezone
        delta = self.data_fim - timezone.now().date()
        return max(delta.days, 0)

    def __str__(self):
        return f'Garantia — {self.usina.nome} ({self.meses} meses)'
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client -> /api/auth/token/ | Credenciais do usuario (username + password) chegam como input nao confiavel |
| client -> /api/* | Header Authorization com Bearer token — validado pelo simplejwt |
| browser -> CORS preflight | Origem da requisicao — validada pelo django-cors-headers |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-1-01 | Repudiation | TokenRefreshView | mitigate | `BLACKLIST_AFTER_ROTATION = True` + `rest_framework_simplejwt.token_blacklist` em INSTALLED_APPS — refresh token anterior e invalidado apos rotacao |
| T-1-02 | Tampering | CorsMiddleware | mitigate | `CORS_ALLOWED_ORIGINS` restrito via env var; `CORS_ALLOW_ALL_ORIGINS` nunca definido; sem wildcard |
| T-1-03 | Elevation of Privilege | TokenObtainPairView | accept | Brute-force no login — rate limiting sera adicionado em fase posterior; risco baixo para admin interno |
| T-1-04 | Information Disclosure | error responses | mitigate | DRF por padrao retorna erros genericos (detail: string); Django DEBUG=False em prod; sem stack traces |
| T-1-05 | Tampering | SECRET_KEY | mitigate | SECRET_KEY lido de env var `DJANGO_SECRET_KEY`; nunca hardcoded (ja praticado no projeto) |
| T-1-06 | Elevation of Privilege | access token | mitigate | `ACCESS_TOKEN_LIFETIME = 15 min` limita janela de uso de token comprometido |
</threat_model>
